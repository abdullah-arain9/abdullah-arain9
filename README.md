### 📐 My Development Approach

```
📁 lib/
├── 🗂️  core/             → Theme, constants, shared utilities
├── 🗂️  data/             → Repositories, models, API layer
│   ├── models/
│   ├── repositories/
│   └── sources/         → Remote (API) & Local (Hive/SQLite)
├── 🗂️  domain/           → Business logic & use cases
├── 🗂️  presentation/     → Screens, widgets, providers
│   ├── screens/
│   ├── widgets/
│   └── providers/
└── 📄  main.dart
```

> I follow **Clean Architecture** — separating UI, business logic, and data so apps stay maintainable at any scale.
